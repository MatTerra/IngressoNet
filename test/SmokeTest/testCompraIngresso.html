Last Document Change:

Status of Document:
Standard

Test purpose

Verificar o funcionamento da função de compras.

Known issues:

  • O botão de decrementar não funciona.

Preconditions of the test:

  • O banco de dados deve estar rodando com o IP local 172.18.0.2 ou o IP do banco deve ser atualizado na classe MySQLHelper.
  • Rodar o programa e fazer login com o usuário 052.327.931-01 senha 123456.
  • Clicar em "Pesquisar jogos" para abrir a página de pesquisa.
  • Selecione "DF" no campo "Estado".
  • Selecione "Todos" no campo "Cidade".
  • Preencha a "01/11/2019" no campo "Início".
  • Clique em "Pesquisar".
  • Clique no botão "Comprar" em algum jogo disponível.

Test documents

  • Nenhum documento é necessário.

Test cases

Test case do botão "+"
Entrada/Procedimento
  1. Clicar no botão "+".
Saída
  • Incrementar quantidade de ingressos.
Test case do botão "-"
Entrada/Procedimento
  1. Clicar no botão "-".
Saída
  • Decrementar quantidade de ingressos.
Test case do botão "Cancelar"
Entrada/Procedimento
  1. Clicar no botão "Cancelar".
Saída
  • Uma janela de aviso aparecerá com a mensagem "Sua compra será cancelada e você voltará para a tela de Pesquisa.".
Test case do botão "Voltar"
Entrada/Procedimento
  1. Clicar no botão "Voltar".
Saída
  • Uma janela de aviso aparecerá com a mensagem "Sua compra será cancelada e você voltará para a tela de Pesquisa.".
Test case do botão "Cancel" da tela de avisos
Entrada/Procedimento
  1. Clicar no botão "Cancelar".
  2. Clicar no botão "Cancel" da janela de avisos.
Saída
  • Fecha a janela de avisos e volta à página de compras.
Test case do botão "x" da tela de avisos
Entrada/Procedimento
  1. Clicar no botão "Cancelar".
  2. Clicar no botão "x" da janela de avisos.
Saída
  • Fecha a janela de avisos e volta à página de compras.
Test case do botão "Ok" da tela de avisos
Entrada/Procedimento
  1. Clicar no botão "Cancelar".
  2. Clicar no botão "Ok" da janela de avisos.
Saída
  • Volta à página de pesquisa.
Test case do botão "Ok" da tela de avisos
Entrada/Procedimento
  1. Preencher quantidade com 1.
  2. Clicar no botão "Comprar".
  3. Confirmar a compra no diálogo aberto.
Saída
  • O programa abre a página de visualização de ingressos. O novo ingresso deve estar disponível.
