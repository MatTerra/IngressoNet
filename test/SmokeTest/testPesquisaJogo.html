Test purpose

Verificar o funcionamento da função de pesquisa.

Known issues:

  • Não existem problemas conhecidos

Preconditions of the test:

  • O banco de dados deve estar rodando com o IP local 172.18.0.2 ou o IP do banco deve ser atualizado na classe MySQLHelper.
  • Rodar o programa e fazer login com o usuário 052.327.931-01 senha 123456.
  • Clicar em "Pesquisar jogos" para abrir a página de pesquisa.

Test cases

Test case todas as partidas a partir de hoje
Entrada/Procedimento
  1. Clique em pesquisar.
Saída
  • Lista de jogos após a data atual. ID's 6 e 7.
Test case todas as partidas a partir de 1/11/2019
Entrada/Procedimento
  1. Altere a data para 1/11/2019 00:00.
  2. Clique em pesquisar.
Saída
  • Lista de jogos após a data 1/11/2019. ID's 3 a 7.
Test case opção de cidades para estados diferentes de DF
Entrada/Procedimento
  1. Abra a lista suspensa de estados clicando no campo "Estado".
  2. Selecione qualquer opção.
  3. Clique para espandir a lista de "Cidade".
Saída
  • Opção "Todos" e a capital do estado devem aparecer. No DF há a Ceilândia além da capital.
Test case pesquisa de jogos por estado
Entrada/Procedimento
  1. Altere a data para 1/11/2019 00:00.
  2. Selecione "DF" no campo "Estado".
  3. Selecione "Todos" no campo "Cidade".
  4. Clique em "Pesquisar".
Saída
  • Lista de jogos após a data 1/11/2019 no DF devem aparecer. ID's 3 e 4.
Test case pesquisa de jogos por cidade e estado
Entrada/Procedimento
  1. Altere a data para 1/11/2019 00:00.
  2. Selecione "DF" no campo "Estado".
  3. Selecione "Brasília" no campo "Cidade".
  4. Clique em "Pesquisar".
Saída
  • Lista de jogos após a data 1/11/2019 no DF devem aparecer. ID 3.
Test case pesquisa de jogos por cidade
Entrada/Procedimento
  1. Altere a data para 1/11/2019 00:00.
  2. Selecione "Todos" no campo "Estado".
  3. Selecione "Ceilândia" no campo "Cidade".
  4. Clique em "Pesquisar".
Saída
  • Lista de jogos após a data 1/11/2019 em Ceilândia devem aparecer. ID 4.
Test case "Voltar"
Entrada/Procedimento
  1. Clique no botão "Voltar".
Saída
  • Volta para a janela "Dashboard".
Test case para "Comprar"
Entrada/Procedimento
  1. Selecione "DF" no campo "Estado".
  2. Selecione "Todos" no campo "Cidade".
  3. Preencha a data "01/11/2019" no campo "Início".
  4. Clique em "Pesquisar".
  5. Clique no botão "Indisponível" em algum jogo.
  6. Clique no botão "Comprar" em algum jogo disponível.
Saída
  • O botão "Indisponível" não deve reagir.
  • Ir para a janela de compras após clicar em comprar.
