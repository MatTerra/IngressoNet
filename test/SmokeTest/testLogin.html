Test Name: Login no Sistema

Document Owner:
Bruno Murta, Mateus Berardo, Thiago Cardoso

Last Document Change:
26. 11. 2019

Status of Document:
Standard

Test purpose

Verificar o funcionamento da página de login.

Known issues:

  • Não existem problemas conhecidos

Preconditions of the test:

  • O banco de dados deve estar configurado com o IP local 172.18.0.2 ou o IP do banco deve ser atualizado na classe MySQLHelper.

Test documents

  • Nenhum documento necessário

Test cases

Test case banco de dados não iniciado
Entrada/Procedimento
  1. Garanta que o programa não tem acesso ao banco de dados.
  2. Preencha o CPF com o valor 052.327.931-01
  3. Digite "123456" no campo senha
  4. Clique em "Entrar".
  5. Inicialize o banco de dados
Saída
  • Um diálogo avisando que houve um erro será exibido. Erro 01 - No Connection to DB.
Test case senha incorreta
Entrada/Procedimento
  1. Preencher o cpf com 052.327.931-01.
  2. Preencher a senha com 123455.
  3. Clicar no botão "Entrar".
Saída
  • Mensagem de Senha incorreta.
Test case usuário inexistente
Entrada/Procedimento
  1. Preencher o cpf com 726.899.300-56.
  2. Preencher a senha com 123456.
  3. Clicar no botão "Entrar".
Saída
  • Mensagem de Usuário não adastrado.
Test case pré-validação cpf e senha
Entrada/Procedimento
  1. Preencher o cpf com 052.327.931-01.
  2. Preencher a senha com 123456.
Saída
  • Botão deve estar inativo até a inserção do último caractere da senha.
Test case login bem sucedido
Entrada/Procedimento
  1. Preencher o cpf com 052.327.931-01.
  2. Preencher a senha com 123456.
  3. Clicar no botão "Entrar".
Saída
  • Programa abre a página de Dashboard.
Test case banco de dados caído
Entrada/Procedimento
  1. Encerre o banco de dados
  2. Preencha o CPF com o valor 000.000.000-00
  3. Digite "123456" no campo senha
  4. Clique em "Entrar".
Saída
  • Um diálogo avisando que houve um erro será exibido.
  • Caso os outros testes tenham sido executados na mesma instância do programa, o erro será Erro 02 - Erro na Query.
  • Caso nenhum acesso a banco de dados tenha sido feito antes, o erro será Erro 01 - No Connection to DB.
Test case acesso à página de cadastro
Entrada/Procedimento
  1. Clicar no botão "Não possuo cadastro".
Saída
  • O programa abre a página de cadastro.
